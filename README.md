# Azure Sales Analytics Platform - End to End Data Warehouse Project


## Objective

Build an enterprise analytics platform using Azure services to transform raw sales data into business insights.


## Tech Stack

- Azure Data Factory
- Azure Data Lake
- Azure SQL Database
- SQL
- Power BI


## Architecture

Data Source
↓
Azure Data Lake
↓
ETL Pipeline
↓
Data Warehouse
↓
Power BI Dashboard
